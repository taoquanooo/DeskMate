# DeskMate 发布验收清单

## 自动门槛

- [ ] `pnpm verify` 通过。
- [ ] `cargo test --manifest-path src-tauri/Cargo.toml` 通过。
- [ ] Windows x64 NSIS 与签名更新产物由 GitHub Actions 生成。
- [ ] 安装包不内置 WebView2，大小不超过 25 MB。
- [ ] 宠物 ZIP 顶层只有 `pet.json`、`spritesheet.webp`、`ASSET_LICENSE.txt`。

## 三台电脑、三个工作日

- [ ] 覆盖 100%、125%、150% DPI，以及双屏/三屏、负坐标和任务栏不同方位。
- [ ] 验证拖动、单击、双击、点击穿透快捷键、托盘暂停和当前屏幕召回。
- [ ] 验证全屏隐藏/恢复、显示器热插拔、休眠唤醒和减少动画。
- [ ] 验证间隔、固定时间、五分钟后提醒、五分钟合并与不补发睡眠历史。
- [ ] 验证离线启动、目录超时、损坏包回退、宠物自动更新和应用重启更新。
- [ ] 连续运行八小时：idle 平均 CPU <2%、内存 <150 MB 且无持续增长。
- [ ] 冷启动首只宠物在三秒内出现。

所有项目通过后发布 `v1.0.0`；否则在 `v0.1.x` 修复并重新计时。
