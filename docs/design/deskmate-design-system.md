# DeskMate UI Design System

Concept reference: `deskmate-settings-concept.png` (generated from the approved DeskMate product plan and the Yanghao v2 pet atlas).

## Visual direction

- Calm, airy Windows utility with a playful mascot rather than a dashboard aesthetic.
- True white application background with a subtle cool-gray preview surface.
- Open split layout: 304 px navigation rail, flexible preview region, flexible settings region.
- No nested card grid, gradients, glass effects, badges, analytics, or decorative filler.

## Tokens

- Canvas: `#ffffff`
- Subtle surface: `#f7f7f6`
- Primary text: `#242322`
- Muted text: `#84817d`
- Divider: `#e9e7e4`
- Accent: `#ff5f38`
- Accent soft: `#fff0eb`
- Success: `#4e9a51`
- Radius: 12 px controls, 18 px preview, 20 px bubble
- Shadows: one low-contrast elevation only for the speech bubble
- Typeface: `Inter`, `Segoe UI Variable`, `Microsoft YaHei UI`, sans-serif

## Component families

- Left navigation rows with one 3 px accent rail and outline icons.
- Open setting rows separated by 1 px dividers.
- Native-looking range controls with coral progress and circular thumb.
- Compact coral/gray toggle switches.
- Speech bubble with white surface, 20 px radius, and a short lower-left tail.
- Primary button: solid coral; secondary button: white with cool-gray border.
